# Pasos para ejecutar

## Clonar el repositorio

```bash
git clone https://github.com/X-R4YM0ND/RetoAngular.git
```

## Instalar dependencias
```bash
npm install
```

## Lanzar JSON Server
```bash
cd api
json-server --watch db.json --port 3100
```
